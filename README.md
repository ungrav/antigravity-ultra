## 🇺🇸 English

**Superpowers for your AI Agents. Change the rules of the game.**

Antigravity Ultra is a portable, ultra-lightweight kernel built as a radical evolution over the original **Google Antigravity** ecosystem. It is designed to manage the identity, persistent memory, and operational capabilities of autonomous AI agents.

No more losing context every time you start a new chat. No more wasting massive amounts of tokens repeating the same instructions. With Antigravity Ultra, every single project gets a unique, persistent, and highly structured memory.

### 🌟 Features

We've completely re-engineered how AI interacts with your projects. Read the full details of all Antigravity Ultra superpowers in our **[FEATURES_EN.md](./FEATURES_EN.md)** guide.

**Highlights:**
- 🧠 Persistent & Structured Memory
- 📐 Zero Monoliths Architecture
- 🔧 On-demand Autofix Skill
- 🕵️ Adversarial Verification
- 📚 Permanent Project Ledgers
- 🛠️ Automatic Skill Discovery

### 📂 The Portable Kit

The "secret" of Antigravity Ultra lies in just 5 core files:

1. **`GEMINI.md` (The Brain):** The operational rulebook. It tells the AI who it is and defines safety boundaries.
2. **`MEMORY.md` (The Hippocampus):** Teaches the AI how to create and read structured project memories.
3. **`GEMINI_BLUEPRINTS.md` (The DNA):** The master rescue file containing templates to restore the system.
4. **`portable-kernel.sh`:** Smart installer for macOS/Linux.
5. **`portable-kernel-windows.ps1`:** Smart installer for Windows PowerShell.

### 🚀 Quick Install

**Forget about manually moving files.** The Antigravity Ultra engine is completely autonomous. Download the 5 portable files to your desired folder and run the installer. You can choose between 4 **Tiers** depending on your needs:

1. **🌱 Minimal:** *Ideal for small projects or quick tests.* Installs only the basic engine, live state, and memory vault. The bare minimum for context without extra tools.
2. **⭐ Recommended:** *The sweet spot for most developers.* Includes everything in Minimal, plus **Advanced Workflows** and **Audit Tools** to automate repetitive tasks and maintain high code quality.
3. **🔥 Complete:** *For hardcore users and massive projects.* Installs the entire ecosystem. Adds MCP templates, deep performance evaluations, telemetry, and core testing. An enterprise-level AI dev environment.
4. **⚙️ Custom:** *You are in control.* Lets you decide step-by-step which modules to install and which to skip.

**macOS / Linux:**
```bash
bash portable-kernel.sh --tier recommended --init
```

**Windows:**
```powershell
.\portable-kernel-windows.ps1 -InstallTier recommended -Init
```

> **Done!** The system will build the hidden `.agent/` folder, inject the initial context, and your favorite AI will be ready to work like a pro from the very first prompt.
